# Chatter and Checkers

An AI-powered web application that helps board game enthusiasts ask questions about games and discover personalized recommendations.

## Features

- **Ask About Games**: Get instant answers to questions about board game rules, mechanics, strategies, and gameplay
- **Find Recommendations**: Receive personalized board game recommendations based on your preferences
- **AI-Powered**: Leverages the Groq API with advanced LLM models for intelligent conversations
- **Clean UI**: Modern, responsive design built with Material-UI
- **Safe & Focused**: AI is constrained to board game discussions only

## Tech Stack

- **Frontend Framework**: React 19 with TypeScript
- **UI Library**: Material-UI (MUI) 7.3
- **Styling**: Emotion (CSS-in-JS)
- **Routing**: React Router DOM
- **AI/LLM**: Groq API
- **Build Tool**: React Scripts 5
- **Code Quality**: ESLint, Prettier, TypeScript

## Prerequisites

- Node.js 16+ and npm
- Groq API key (get one at [console.groq.com](https://console.groq.com))
